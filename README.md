## V10.0 Release 3｜AI 租賃中心測試版
- 新增一般民眾／法人承租
- 新增已知租金正推、已知屋主實拿反推
- 新增所得稅預扣、補充保費、營業稅可調設定

- 新增房屋稅、地價稅、管理費、修繕與其他持有成本（可選）
- 分開顯示「屋主每月實拿」與「扣除持有成本後淨收入」
- 新增租賃分析分享設定

### 稅務提醒
本功能屬快速概估工具。所得稅扣繳、補充保費、營業稅與發票適用，仍須依出租人、承租人、租約及稅籍狀況確認。


## V10.0 Release 4｜AI 租賃中心重製版
- 承租人為自然人：預設不扣租金所得稅與補充保費
- 公司行號給付租金：所得稅依 10% 計算，應扣稅額不超過 2,000 元時免扣
- 個人房東租金單次給付達 20,000 元：預設納入 2.11% 補充保費
- 營業稅改為人工確認，不因店面或法人承租直接強制套用
- 新增年度所得申報提醒：扣繳屬預繳；房屋租金43%法定可扣除費用標準（僅土地出租不適用）
- 稅務規則可由進階模式調整


## V10.0 Release 5｜租賃雙方服務費測試
- 房東服務費與租客服務費分開設定
- 預設：房東 1 個月、租客 0.5 個月
- 雙方均支援 0／0.5／1／1.5／2 個月及固定金額
- 顯示仲介服務費合計
- 顯示租客簽約準備金
- 房東服務費可一次扣除或攤提至租期


## V10.0 Release 6｜服務費獨立計算
- 房東與租客仲介服務費仍分開計算
- 仲介服務費不再扣除屋主每月租金實拿
- 仲介服務費不再扣除持有成本後淨收入
- 簽約當下屋主租金現金流不扣仲介費
- 房東服務費、租客服務費、仲介費合計改為獨立列示


## V10.0 Release 7｜租金門檻修正
- 20,000 元整：所得稅預扣為 0，補充保費為 422 元
- 20,001 元以上：所得稅與補充保費皆納入
- 所得稅判斷依原始 10% 稅額是否「超過 2,000 元」
- 補充保費依單次租金是否達 20,000 元以上
- 修正瀏覽器可能自動恢復進階勾選狀態的問題

## V10.0 Release 8｜租賃步驟式介面

## V10.0 Release 10｜門檻說明修正
